# vmbnet_async_two_cameras_viewer - 双相机自由采集C#例程
Vimba .NET C# 异步采集图像例程 - 支持两个相机同时运行 - 可以支持Free Run 自由采集模式 以及 硬件触发模式。  
Vimba .NET C# sample for two cameras based on \VimbaNET_Examples\AsynchronousGrab\Forms
